BoostCourse Reservation Project

개발환경
- Win 10
- JAVA (JDK 1.8.0)
- 웹서버: 톰캣 8.5
- 프레임워크: 스프링 4.3.5
- DBMS: mysql
- 기술: JDBC, pure javascript, ajax 등

주요 기능
- 네이버의 전시/공연 예약서비스 구현
- 예약하기 (input: 정규식 유효성검사)
- 로그인하기 (이메일 조회, 로그인 세션 유지)
- 예약취소
- 예약 리뷰쓰기 (접속 조건에 따라 리뷰작성, 예약정보 없음, 이미 등록된 리뷰 등)
- 파일첨부
